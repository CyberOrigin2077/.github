### ◈ What we command

| | |
|---|---|
| **⬡ CAPTURE** | Wearable, first-person rigs and on-device software — **engineered in-house**, run at fleet scale. The physical layer everyone else outsources, we own outright. |
| **⬡ OPERATE** | A platform that decides *what* to capture, drives collection **task by task**, and tracks every one to delivery. Raw human effort, forged into an industrial machine. |
| **⬡ PROCESS** | An engine that smelts raw recordings into structured, queryable data — sampling, **multi-model inference**, indexing, storage, full provenance. |
| **⬡ REFINE** | Automated QC **and** human-in-the-loop review. Nothing ships until it's interrogated to the frame. |
| **⬡ UNDERSTAND** | A 3D perception stack that rips **people, hands, objects, and scenes** out of raw footage — the geometry of action itself. |
| **⬡ ORGANIZE** | A structured map of every real-world scenario worth capturing. We hunt **coverage, not noise** — we know exactly what reality is missing. |
| **⬡ BUILD** | Developer and researcher tooling, plus **open frameworks** — so the world can build on what we mine. |

> Every stage interoperates. One orchestration backbone runs the whole chain.
> No silos. No dead ends. **One system — atoms to intelligence.**
